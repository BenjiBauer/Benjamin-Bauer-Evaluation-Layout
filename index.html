Please prepare the following exercise for your next interview.

Description:

The aim of this exercise is to assess your capabilities of interpreting a design into code.

Use this whole page as the design and content to create a functional styled web page using at least the following technologies: HTML, CSS (Compiled from SCSS) and JavaScript/Typescript.

Feel free to use any frameworks and 3rd party libraries.

The page needs to responsive for mobile screens so the menu and content will need to adjust accordingly.

Get creative in how you see the elements of the design working… (e.g. a sticky header, button animations etc…)

We will assess each candidate based on how accurate the final output is, code quality and technologies used.
